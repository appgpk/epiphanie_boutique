# Shoe Preference Experiment — Epiphanie Boutique

## Project Overview

This project implements a **simple preference experiment** designed to understand customer tastes for shoes in **Burkina Faso**.

Instead of using a long survey, we created a **very simple interaction** where participants only need to click:

- 👍 **I like**
- 👎 **I don't like**

This approach reduces survey fatigue and allows us to collect **rapid feedback on many products**.

The goal is to identify which shoes customers prefer before importing new inventory.

---

## Motivation

Collecting customer preferences using traditional surveys is difficult because:

- long questionnaires discourage participation
- response rates decrease quickly
- customers may not complete the survey

To solve this problem, we designed a **minimal interaction experiment** where users only need to express a simple preference.

This allows us to:

- collect more responses
- reduce cognitive effort for participants
- obtain clearer signals about product preferences.

---

## How the Experiment Works

Participants are shown a sequence of **shoe images**.

For each product they answer one question:

> Do you like this shoe?

They respond by clicking one of two buttons:

- 👍 **I like**
- 👎 **I don't like**

Each response is recorded as a binary preference.
